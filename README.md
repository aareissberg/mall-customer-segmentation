# mall-customer-segmentation

This project applies unsupervised machine learning techniques to segment mall customers based on purchasing behavior and demographics. The goal is to identify distinct customer groups that can inform targeted marketing strategies.

## Methodology
This analysis includes:
- Exploratory Data Analysis (EDA)
- Feature exploration and visualization
- K-Means clustering for customer segmentation
- Principal Components Analysis for visualizing customer segments in reduced space
- Cluster interpretation for marketing insights

## Dataset
The dataset contains information on mall customers including:
- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

## Results
The clustering model identified several distinct customer segments characterized by different spending patterns and income levels.

These segments can support targeted marketing strategies for improving engagement and revenue.

Segments and Marketing Initiatives:
- **Low Income + Low Spend**: Offer entry-level loyalty programs or bundled value deals to increase visit frequency
without pressuring high spending.
- **Low Income + High Spend**: Target with flash sales and rewards for continued engagement.
- **High Income + High Spend**: Introduce premium experiences, VIP programs, or early access events — they are
your core profit drivers and respond to personalized, high-status offers.
- **High Income + Low Spend**: Use personalized promotions or premium product sampling to increase conversion
— they have capacity to spend more but need better targeting.
